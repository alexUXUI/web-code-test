User stories

  • As a consumer, I would like to be able to browse a list of article categories so I can find articles I'm interested in reading.
  • As a consumer, I would like to see a list of articles for a category I have selected so that I'm only presented with articles I care about. As to not get overwhelmed, I would only like to see a listing of 5 articles at a time.
  • As a consumer, I would like to click on an article and see it's full contents so I can easily read the article.
  • As a consumer, I would like some indication that I have read the article so I don't try to read the same article again.
  • As a consumer, I would like to submit new article requests
